Tags semânticas do HTML

ㅤ Algumas tags consideradas importantes dentro de uma estrutura HTML básica.

•ㅤ<header>

ㅤA tag header representa uma espécie de conainer para as informações introdutórias da página. Geralmente nessa tag é onde vai o logo ou o icon, informações sobre o autor, barra de navegação, etc.

•ㅤ<section>

ㅤA tag section representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo.

•ㅤ<article>

ㅤA tag article representa o post de um fórum, um artigo de revista ou jornal, um post de um blog, um comentário enviado por um usuário, um gadget ou widget interativos, ou qualquer outra forma de conteúdo independente.

•ㅤ<nav>

ㅤA tag nav representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.

•ㅤ<aside>

ㅤA tag aside é geralmente utilizada em propagandas ou mini navegadores que ficam na parte lateral do site.

•ㅤ<main>

ㅤA tag main serve para separar o conteúdo principal do site das demais partes do site.

•ㅤ<figure>

ㅤA tag figure é específica para quando se coloca uma imagem, ilustração, diagrama, fotos, etc.

•ㅤ<footer>

ㅤA tag footer é a parte onde fica o rodapé do site, onde geralmente ficam informações como autoria, direitos autorais, contatos, mapa do site, documentos relacionados, etc.

•ㅤ<a>

ㅤA tag a normalmente é usada para ligar links de outras paginas dentro do site.

•ㅤ<em>

ㅤA tag em é usada para formatar o texto como itálico.

•ㅤ<strong>

ㅤA tag strong é usada para formatar o texto como negrito.

•ㅤ<cite>

ㅤA tag cite representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência, que pode ser em uma forma abreviada de acordo com as convenções usadas para a adição dos metadados de citação.

•ㅤ<q>

ㅤA tag q normalmente inserem aspas ao redor da citação.

•ㅤ<time>

ㅤA tag time é usado para traduzir a hora em um formato legível por máquina para que os navegadores possam adicionar lembretes de data por meio do calendário do usuário e os mecanismos de pesquisa possam produzir resultados de pesquisa mais inteligentes.

•ㅤ<address>

ㅤA tag address define as informações de contato do autor/proprietário de um documento ou artigo.
